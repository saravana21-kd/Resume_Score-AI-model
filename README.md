<p id="description">The Resume Score AI Model helps users evaluate how well their resume aligns with a specific job description. It uses text similarity techniques to calculate a score and identify how closely a candidate’s profile matches job requirements.</p>



  
<h2>🧐 Features</h2>

Here're some of the project's best features:

*   📄 Upload resume and job description
*   🤖 AI-based similarity scoring
*   📊 Instant match percentage output
*   ⚡ Fast and simple interface

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/your-username/Resume_Score-AI-model.git cd Resume_Score-AI-model
```

<p>2. Create virtual environment (recommended)</p>

```
python -m venv venv 
```

```
venv\Scripts\activate # Windows
```

<p>3. Install dependencies</p>

```
pip install -r requirements.txt
```

<p>4. Run the application</p>

```
python app.py
```

<p>5. Open in browser</p>

```
http://127.0.0.1:5000/
```
